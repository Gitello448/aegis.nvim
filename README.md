## Install

### Lazy

```lua
{
  'aekasitt/aegis.nvim',
  cmd = 'AegisToggle',
}
```

## Usage

`AegisToggle`

```lua
{
  'aekasitt/aegis.nvim',
  cmd = 'AegisToggle',
  opts = {
    timeout = 1,
    maxkeys = 5,
    -- more opts
  }
}
```
